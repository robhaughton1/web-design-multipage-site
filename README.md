# Web Design Multipage Site

This project is a multi‑page HTML and CSS website created as part of an early web development assignment, representing a fictional web‑design company named after me and includes a homepage, an About page, and a Services page, all styled with a shared external stylesheet.

## Features
- **Three‑page website** (Home, About, Services)
- **Consistent navigation bar** across all pages
- **External CSS stylesheet** for layout and design
- **Organized image folder** for assets
- **Clean, readable HTML structure**

## What I practiced
- Building a multi‑page website
- Creating and styling navigation menus
- Structuring content with semantic HTML
- Applying consistent styling across multiple pages
- Organizing project files (HTML, CSS, images)

## Technologies Used
- HTML5
- CSS3
